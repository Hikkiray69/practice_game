# TASKLIST — Interactive Learning Game

Этот чеклист используется как рабочий план проекта: от идеи до сдачи.

## Как пользоваться

- Идем сверху вниз по этапам.
- В каждом этапе отмечаем `[x]` только полностью завершенные пункты.
- Не переходить к следующему этапу, пока не закрыты критичные пункты текущего.
- После каждого этапа делать короткий итог: что сделано, что блокирует, что дальше.
