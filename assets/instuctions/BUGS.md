# BUGS

Журнал багов проекта. Ведется в формате: `id`, статус, описание, влияние, решение.

## Открытые

- Пока нет открытых багов.

## Решенные

- **BUG-001** (resolved, 2026-04-07)  
  **Проблема:** перемещение игрока на `WASD` не работало при русской раскладке клавиатуры.  
  **Влияние:** пользователь не мог двигаться через базовую схему управления в non-EN layout.  
  **Решение:** обработка клавиш переведена с `event.key` на layout-independent `event.code` (`KeyW/KeyA/KeyS/KeyD`, `Arrow*`) в `game/src/features/movement/ui/PlayerAvatar.tsx`.

- **BUG-002** (resolved, 2026-04-07)  
  **Проблема:** при появлении вариантов ответов возникал скролл экрана.  
  **Влияние:** ухудшение UX, "прыгающий" layout во время выбора.  
  **Решение:** для блока вопросов введены ограничения высоты и внутренний скролл (`.panel { max-height; overflow-y }`) в `game/src/app/globals.css`.

- **BUG-003** (resolved, 2026-04-07)  
  **Проблема:** в тексте AI-подсказки отображался технический хвост `status: inProgress`.  
  **Влияние:** неестественный текст и шум в пользовательском интерфейсе.  
  **Решение:** убрано включение статуса в fallback-текст AI в `game/src/app/api/ai/assist/route.ts`.
